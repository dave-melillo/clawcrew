# Research Template

Use when: User wants to understand something

## Workflow

1. **Search** - Use available tools (web_search, x-research, web_fetch)
2. **Synthesize** - Summarize findings, cite sources
3. **Recommend** - Give actionable takeaways

## Tools Priority

1. X research skill (for tech/dev topics)
2. web_search (for general topics)
3. web_fetch (for specific URLs)
4. Memory search (for past context)

## Output Format

### Summary
[2-3 sentences]

### Key Findings
- Finding 1 (Source)
- Finding 2 (Source)

### Recommendation
[What to do with this info]
