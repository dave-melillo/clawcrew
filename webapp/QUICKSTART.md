# üöÄ ClawCrew Quick Start

Get your AI crew running in 3 minutes!

## Step 1: Install & Run (2 minutes)

```bash
cd /home/clawdbot/clawd/clawcrew/webapp

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open http://localhost:3000 in your browser.

## Step 2: Build Your First Crew (5 minutes)

### Add Agents

1. You'll see 8 agent templates on the home page
2. Click **"Add to Crew"** on 2-3 agents (e.g., The Boss, The Builder, The Brain)
3. Each agent card shows:
   - üé≠ Emoji and name
   - Role and vibe
   - Default AI model

### Customize an Agent (Optional)

1. Click **"Edit"** on any agent
2. You can:
   - Change the name
   - Edit the SOUL.md (personality/instructions)
   - Select a different AI model
   - Adjust temperature (creativity level)
   - Add routing keywords
3. Click **"Save Changes"**

## Step 3: Connect Telegram (5 minutes)

1. Click **"Channels"** in the top nav
2. Click on the **Telegram** card
3. Follow the 3-step wizard:
   - **Step 1:** Instructions for creating a bot with @BotFather
   - **Step 2:** Enter your bot token
   - **Step 3:** Success! Your bot is connected

## Step 4: Test It Out!

1. Open Telegram and search for your bot's username
2. Send a message like "Hello!" or "Can you research AI agents?"
3. Your agents will respond based on their personalities and routing
